# Basic of Programming

This folder covers the fundamental programming concepts needed before starting DSA. Topics are organized into separate folders, each focused on one concept.

---

## Topics Covered

| Topic | What it covers |
|-------|----------------|
| Input-Output | Taking input from user and printing output |
| Data Types | Integer, float, string, boolean, and type sizes |
| If Else Statements | Conditional logic, nested conditions, elif chains |
| Switch | Switch-case equivalent in Python and C++ |
| Loops | For loop, while loop, do-while, nested loops |
| Function | Defining functions, parameters, return values, scope |
| Array | 1D and 2D arrays, traversal, insertion, deletion |
| String | String operations, slicing, built-in methods |

---

## Folder Details

**Input-Output** — How to read input from the user and display output. Covers formatted printing and multiple input methods.

**Data Types** — Understanding the different types of data a variable can hold and how memory is allocated for each type.

**If Else Statements** — Decision-making in programs. Covers simple if, if-else, elif, nested conditions, and short-hand ternary style.

**Switch** — Multi-way branching. In C++ it uses `switch-case`, in Python it can be done using `match-case` (Python 3.10+) or if-elif chains.

**Loops** — Repeating a block of code. Covers all loop types with break and continue usage, and nested loop patterns.

**Function** — Reusable blocks of code. Covers function definition, calling, arguments, default values, and return statements.

**Array** — Collection of elements stored at contiguous memory. Covers 1D arrays, 2D arrays, traversal, and basic operations.

**String** — Sequence of characters. Covers string traversal, comparison, slicing, reversal, and commonly used built-in methods.
