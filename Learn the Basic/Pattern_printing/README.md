# Pattern Printing

This folder covers all types of pattern printing problems. Pattern printing helps in building a strong understanding of nested loops, which is a core skill for DSA.

All patterns are implemented in both Python and C++.

## Pattern Categories

**Star Patterns** — Use `*` to build shapes like triangles, pyramids, diamonds, and rectangles. Focus is on controlling spaces and stars in nested loops.

**Number Patterns** — Replace stars with numbers. Numbers can increase row by row, column by column, or follow a binary sequence.

**Alphabet Patterns** — Use characters A, B, C... in place of numbers or stars. Based on ASCII value manipulation.

**Special Patterns** — Combinations of the above, like symmetric void or crown patterns that require careful spacing logic.
